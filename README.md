**AI Agent with YouTube Integration using Model Context Protocol (MCP)**
🔗 **Live Website:** https://ai-agent-wko1.vercel.app/
This is the fully deployed AI Agent interface where users can authenticate with Google and interact with the YouTube-connected MCP agent.
This project is a fully deployed AI-powered agent that interacts with YouTube using the Model Context Protocol (MCP).
The agent understands user queries, selects appropriate YouTube actions through MCP tools, and returns personalized results such as watch history, liked videos, or recommendations.
🚀 Key Features
🤖 AI Agent Powered by OpenAI
🔗 Model Context Protocol (MCP)
🎥 YouTube API Integration
🔐 Google OAuth Login
🌐 Full Deployment
